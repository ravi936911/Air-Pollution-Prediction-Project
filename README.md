# Air Pollution Analysis and Prediction using Machine Learning

This project focuses on analyzing air pollution data and predicting average pollution levels using Machine Learning techniques. The dataset is cleaned, preprocessed, normalized, and visualized to understand pollution trends and patterns. A Linear Regression model is used to predict pollutant average values based on different features.

## Features

* Data cleaning and handling missing values
* Data normalization using MinMaxScaler
* Label Encoding for categorical columns
* Statistical analysis (Mean, Median, Variance, Correlation)
* Data visualization using Matplotlib and Seaborn
* Scatter plots, Histogram, Heatmap, Boxplot
* Top 10 polluted cities analysis
* Linear Regression model training and prediction
* Model evaluation using MAE, MSE, RMSE, and R2 Score

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Machine Learning Model

Linear Regression is used to predict the average pollution level from available dataset features.

## Project Goal

To analyze environmental air quality data and build a predictive model that helps understand pollution levels for future planning and decision-making.

## Output

The project generates graphs, statistical insights, and predicted pollution values with performance accuracy metrics.
